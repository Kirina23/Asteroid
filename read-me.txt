Документация по проекту "Мониторинг астероидов" 
Проект "Мониторинг астероидов" разработан для отслеживания и анализа данных об астероидах, 
проходящих близко к Земле. Этот проект предоставляет инструменты для получения информации о 
ближайших астероидах, а также создания списка астероидов, которые могут представлять потенциальную опасность.
Основные функции 
Получение данных о ближайших астероидах от NASA. Формирование списка астероидов с детальной информацией. 
Анализ потенциальной опасности астероидов. Установка и настройка Установка зависимостей 
Для установки и запуска проекта вам понадобятся следующие зависимости:
Node.js (версия 14 или выше) npm (Node Package Manager) Чтобы установить зависимости, 
выполните следующую команду: npm install Сборка и запуск 
Сборка проекта Для сборки проекта используйте команду: npm run build 
Запуск проекта Чтобы запустить проект в режиме разработки, выполните: npm run dev 
После успешной сборки и запуска, приложение будет доступно по адресу http://localhost:3000. 
Использование Получение списка астероидов Откройте приложение в браузере. Перейдите на страницу "Список астероидов".
Нажмите кнопку "Получить список" для загрузки данных о ближайших астероидах. Анализ астероидов 
Выберите астероид из списка. На странице деталей астероида вы найдете информацию о его размере, расстоянии до Земли и потенциальной опасности.
